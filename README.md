# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현할 기능 목록
### Domain 로직
- [x] 랜덤 번호 선정 기능
  - [x] 1 ~ 9 사이의 3자리 숫자 
  - [x] 중복되지 않는 숫자
- [x] 플레이어 숫자 검증 기능
  - [x] 검증 실패시 고객에게 메시지 전달 후 게임 계속 진행 
    - [x] 1 ~ 9 사이의 숫자 인지 검증
    - [x] 3자리의 숫자인지 검증
    - [x] 중복된 숫자가 있는지 검증
- [x] 숫자 판독 기능
  - [x] 같은 수가 같은 자리 : 스트라이크
  - [x] 같은 수가 다른 자리 : 볼
  - [x] 같은 수가 없으면 : 낫싱
